# 푸트맵 프로젝트

## 프로젝트 소개
```
푸드트럭을 찾고 싶어하는 소비자의 니즈와 자신의 푸드트럭을 홍보하고
싶어하는 공급자의 니즈를 충족하기 위한 프로젝트
```

## 컴퓨터 구성 / 필수 조건 안내 (Prerequisites)
  * Node
  * vue-cli

## 설치 안내 (Installation Process)

### 프로젝트 설치
```
npm install
```

### Json Server 설치 (글로벌 설치)
```
npm install -g json-server
```

## 사용법 (Getting Started)

### Json Server DB 구동
```
npm run api
```

### 프로젝트 실행
```
npm run serve
```
